Series Variables

Las series variables son una herramienta estadística utilizada para analizar y predecir el comportamiento de una variable que cambia con el tiempo o bajo diferentes condiciones. Son esenciales en economía, Se emplea para liquidar deudas o acumular fondos a corto, mediano, largo plazo o indefinidamente. Cada flujo sigue una regla específica, variando en una cantidad fija o un porcentaje en relación al flujo anterior, fenómeno conocido como Gradiente.

Las usamos en escenarios de la vida diaria donde podemos encontrar que no siempre las cuotas son fijas. Por ejemplo al adquirir una vivienda se puede pactar el pago con cuotas variables que van aumentando o cuotas variables que van disminuyendo.

Tipos de gradientes

  1. Gradiente lineal creciente: En un gradiente lineal creciente, los flujos de efectivo aumentan de forma constante en cada período.
  2. Gradiente lineal decreciente: En un gradiente lineal decreciente, los flujos de efectivo disminuyen de forma constante en cada período.
  3. Gradiente geométrico creciente: En un gradiente geométrico creciente, los flujos de efectivo aumentan a una tasa(%) constante.
  4. Gradiente geométrico decreciente: En un gradiente geométrico decreciente, los flujos de efectivo disminuyen a una tasa(%) constante.

1.1 Formula valor presente en una gradiente lineal creciente

V P = A × [ ( 1 + i ) n − 1 i × ( 1 + i ) n ] + G i × [ ( 1 + i ) n − 1 i × ( 1 + i ) n − n ( 1 + i ) n ]
  • VP: Valor presente
  • G : Valor Gradiente
  • A : Monto de la primera couta
  • i : Valor de la tasa de interés por periodo.
  • n : Numero total de periodos.

1.2 Formula valor futuro en una gradiente lineal creciente

V F = A × [ ( 1 + i ) n − 1 i ] + G i × [ ( 1 + i ) n − 1 i − n ]
  • VF: Valor futuro
  • G : Valor Gradiente
  • A : Monto de la primera cuota
  • i : Valor de la tasa de interés por periodo.
  • n : Numero total de periodos.

1.3 Formula para calcular el monto de la primera cuota en una gradiente lineal creciente

A = V F − G i × [ ( 1 + i ) n − 1 i − n ] [ ( 1 + i ) n − 1 i ]
  • VF: Valor futuro
  • G : Valor Gradiente
  • A : Monto de la primera cuota
  • i : Valor de la tasa de interés por periodo.
  • n : Numero total de periodos.

1.4 Formula para calcular valor de la cuota en n en una gradiente lineal creciente

C n = A + ( n - 1 ) * G
  • C : Valor cuota en el periodo n
  • G : Valor gradiente
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular

2.1 Formula valor presente en una gradiente lineal decreciente

V P = A × [ ( 1 + i ) n − 1 i × ( 1 + i ) n ] - G i × [ ( 1 + i ) n − 1 i × ( 1 + i ) n − n ( 1 + i ) n ]
  • VP: Valor presente
  • G : Valor Gradiente
  • A : Monto de la primera couta
  • i : Valor de la tasa de interés por periodo.
  • n : Numero total de periodos.

2.2 Formula valor futuro en una gradiente lineal decreciente

V F = A × [ ( 1 + i ) n − 1 i ] - G i × [ ( 1 + i ) n − 1 i − n ]
  • VF: Valor futuro
  • G : Valor Gradiente
  • A : Monto de la primera cuota
  • i : Valor de la tasa de interés por periodo.
  • n : Numero total de periodos.

2.3 Formula para calcular el monto de la primera cuota en una gradiente lineal decreciente

A = V F + G i × [ ( 1 + i ) n − 1 i − n ] [ ( 1 + i ) n − 1 i ]
  • VF: Valor futuro
  • G : Valor Gradiente
  • A : Monto de la primera cuota
  • i : Valor de la tasa de interés por periodo.
  • n : Numero total de periodos.

2.4 Formula para calcular valor de la cuota en n en una gradiente lineal decreciente

C n = A - ( n - 1 ) * G
  • C : Valor cuota en el periodo n
  • G : Valor gradiente
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular

3.1 Formula para calcular valor presente en gradiente geometrico creciente

VP = A × [ ( 1 + J ) n - ( 1 + i ) n ( J - i ) ( 1 + i ) n ]
  • VP : Valor presente n
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular
  • J : Gradiente (%)
  • i : Tasa de interes

3.2 Formula para calcular valor futuro en gradiente geometrico creciente

VF = A × [ ( 1 + i ) n - ( 1 + J ) n ( i - J ) ]
  • VF : Valor futuro
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular
  • J : Gradiente (%)
  • i : Tasa de interes

3.3 Formula para calcular valor de la primera cuota en gradiente geometrico creciente

A = VF × [ ( i - J ) ( 1 + i ) n - ( 1 + J ) n ]
  • VF : Valor futuro
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular
  • J : Gradiente (%)
  • i : Tasa de interes

3.4 Formula para calcular valor de la cuota n en gradiente geometrico creciente

C n = A x ( 1 + J ) n − 1
  • C : Valor cuota en el periodo n
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular
  • J : Gradiente (%)
  • i : Tasa de interes

4.1 Formula para calcular valor presente en gradiente geometrico decreciente

VP = A × [ ( 1 + i ) n − ( 1 - J ) n ( J + i ) ( 1 + i ) n ]
  • VP : Valor presente n
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular
  • J : Gradiente (%)
  • i : Tasa de interes

4.2 Formula para calcular valor futuro en gradiente geometrico decreciente

VF = A × [ ( 1 + i ) n - ( 1 - J ) n ( J + i ) ]
  • VF : Valor futuro
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular
  • J : Gradiente (%)
  • i : Tasa de interes

4.3 Formula para calcular valor de la primera cuota en gradiente geometrico decreciente

A = VP × [ ( J + i ) × ( 1 + i ) n ( 1 + i ) n - ( 1 - J ) n ]
  • VP : Valor presente
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular
  • J : Gradiente (%)
  • i : Tasa de interes

4.4 Formula para calcular valor de la cuota n en gradiente geometrico decreciente

C n = A x ( 1 - J ) n − 1
  • C : Valor cuota en el periodo n
  • A : Valor de la primera cuota
  • n : Numero del periodo a calcular
  • J : Gradiente (%)
  • i : Tasa de interes
